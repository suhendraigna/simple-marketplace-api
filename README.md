# Simple Marketplace API

Backend REST API untuk toko online menggunakan Django & Django REST Framework.

## Scope
- Products & Categories
- Cart
- Orders
- Inventory (basic)

## Architecture
- API-first
- Service layer for business logic
- Transactional checkout flow
- Snapshot-based order items

## Tech Stack
- Python
- Django
- Django REST Framework
- PostgreSQL (planned)
